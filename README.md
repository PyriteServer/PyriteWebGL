# PyriteWebGL

Prototype WebGL client for [Pyrite3d](http://www.pyrite3d.org/)

- Uses [ThreeJS](https://threejs.org/)




## Install

```
npm install --save pyrite3dwebgl
```


## Usage

```js
import Pyrite from 'pyrite3dwebgl';

const pyrite = new Pyrite(container, id, config);
pyrite.start();
```
